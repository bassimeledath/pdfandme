pdfandme · Design mockups · v0.2 · July 2026

Quiet software for loud paperwork.

A PDF editor with five visible tools, one primary button, and no server. Everything Adobe charges $29/month for that people actually use — and nothing they don't.

PRINCIPLE 01

One action per screen

The start page does exactly one thing. The editor has exactly one primary button. Nothing competes for attention.

PRINCIPLE 02

Tools appear when needed

Five tools visible, the rest behind “more.” Options show up next to the thing you're editing — never in a permanent settings panel.

PRINCIPLE 03

Nothing leaves the tab

Privacy stated plainly on every screen, in words a human would use. It's the product's spine, not a footnote.
